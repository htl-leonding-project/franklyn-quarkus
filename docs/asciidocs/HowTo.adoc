== Start the backend first

* open a terminal in `backend`-project and execute:

[source, shell]
----
./mvnw clean quarkus:dev
----

////

== Start the frontend

* open a terminal in `franklyn-frontend`-project and execute:

[source, shell]
----
npm i --force
ng serve --open
----
